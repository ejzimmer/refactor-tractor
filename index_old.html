There are two important principles to remember with grid. First, grid uses the idea that was initially introduced with flexbox, of a container element which controls the layout of its children. You apply the grid property to the parent element, but it's the child elements which are affected. The second is the fact that the grid exists independent of its children. If you're familiar with HTML and CSS, this can be a bit hard to wrap your head around because _that's not how it works normally_
